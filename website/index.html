Get started

Set up Bracker in minutes

1. Install Bracker

npm install -g bracker

2. Configure Claude Code

Add to your Claude Code MCP configuration:

{
  "mcpServers": {
    "bracker": {
      "command": "bracker",
      "args": []
    }
  }
}

3. Initialize in Your Project

cd your-project
bracker init
# Follow the setup wizard
